Yes, we are building the future of the web at **Supernova Labs**. ### 🍿 An interconnected community
The open-source spirit is the 💗 heart of Supernova Labs and fundamental to how we build Gravity-Lang today. See for yourself:
* **Gravity-Lang** has helped developers write blazing-fast physics simulations with a clean, readable DSL 🚀
* Our community is proving that you don't need decades of experience to create tools that impact the ecosystem 🎸
* **Every minute** a developer realizes that modern Vanilla JS APIs are better than 15-year-old wrappers 🏄

Now that we are talking about the important things, ☝️ are you still running slow, bloated simulations in 2026? Yes? It's okay, we can help you get started with **Gravity-Lang**! Open source is built by people just like you. Learn more about <a href="https://github.com/SupernovaxLabs/Gravity-Lang">how to contribute to Gravity-Lang</a>.
